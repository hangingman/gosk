
# Sample

* コメント, `;` か `#`

```
; comment
# comment
```

* ラベル, 英数小文字とアンダーバーを許容

```
entry:
```

* 文, ニーモニックは値を返したりしないのですべて文扱い
  * オペランドは複数の引数をとったりするだろう

```
DB  0x90
MOV  BX, 15
```

* 変数(EQU)

```
BOTPAK  EQU  0x00280000
```

* コンフィグ宣言
  * `[... ""]` の形で設定を追加する

```
[FORMAT "WCOFF"]
```
