# Gosk

It's a play-ground

## Build & Run

* You need to install Go and CMake

```
$ go get github.com/pointlander/peg               # only first time
$ git clone https://github.com/HobbyOSs/gosk.git  # clone
$ cd gosk
$ mkdir build
$ cmake -G Ninja ..
$ ninja
$ ./gosk xxx.nas                                  # read nask file
```
